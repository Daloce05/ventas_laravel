# Taller - Empleados y Ventas (PHP, PSR-4)

Proyecto demo que realiza operaciones sobre arreglos de empleados y ventas, usa PSR-4 autoload con Composer, y contiene un diseño con Bootstrap.

Requisitos:
- PHP 7.4+ o 8.x
- Composer

Instalación:

1. Desde la raíz del proyecto (donde está `composer.json`):

```powershell
composer install
```

2. Ejecutar servidor de desarrollo de PHP (opcional):

```powershell
php -S localhost:8000 -t public
```

Abrir en el navegador: http://localhost:8000

Notas:

- Las vistas usan Bootstrap.
- Se proporcionan formularios para pegar JSON con empleados y ventas, o usar los ejemplos incluidos en la página.

Siguientes pasos (local):
- Ejecutar composer install.
- Probar formularios y revisar resultados.
